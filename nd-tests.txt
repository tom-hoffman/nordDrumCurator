Saving & loading
----------------

A reference for implementing this in the main code later.

    >>> import tempfile
    >>> import pickle
    >>> t = tempfile.TemporaryFile()
    >>> t.write(pickle.dumps(tdr)) # output not significant
    7081
    >>> t.seek(0) 
    0
    >>> loaded = pickle.load(t)
    >>> print(loaded)
    DataRoot has 30 programs on disc and 30 in memory.
    >>> t.close()
